# DnD Tools

A web-based toolset for Dungeons & Dragons, built to support tabletop gameplay with interactive maps, character management, and session utilities.

## Features

- Interactive maps powered by Leaflet
- User authentication via NextAuth
- PostgreSQL database with Prisma ORM
- Responsive UI with Material UI and Framer Motion
- Data tables and forms via TanStack

## Tech Stack

- **Next.js 16** — React 19 with server components
- **TypeScript** — end-to-end type safety
- **Prisma** — database ORM with PostgreSQL
- **NextAuth** — authentication
- **Leaflet** — interactive maps
